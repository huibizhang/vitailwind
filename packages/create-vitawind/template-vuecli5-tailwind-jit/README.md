# template-vuecli5-tailwind-jit

- **template of:** vue-cli 5-beta + vue3
- **postcss ver:** 8
- **tailwindcss:** v2.2.7

## Started
with npm:
```bash
npm install
```
with yarn:
```bash
yarn
```

---
## Develop
with npm:
```bash
npm run serve
```
with yarn:
```bash
yarn serve
```

---
## Build
with npm:
```bash
npm run build
```
with yarn:
```bash
yarn build
```
